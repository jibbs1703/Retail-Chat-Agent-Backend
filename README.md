# Research Paper Agent

An intelligent, automated system designed to streamline the entire research paper workflow — from discovery to citation management.

## Overview

This project aims to build an AI-powered research assistant that helps academics, students, and researchers efficiently navigate and synthesize scholarly literature.

## Features

### Academic Search
- Search across multiple academic databases and sources
- Filter and rank papers by relevance, citation count, and recency
- Track and organize search queries

### Paper Analysis
- Extract key insights, methodologies, and findings
- Summarize complex research papers
- Identify relationships between papers

### Literature Review Generation
- Generate structured literature reviews
- Synthesize findings across multiple papers
- Create comparative analyses

### Citation Management
- Automatic citation extraction and formatting
- Support for multiple citation styles (APA, MLA, Chicago, etc.)
- Reference organization and deduplication

## Tech Stack

- **Frontend**: Gradio
- **Backend**: Python
- **LLM**: Open Source Models from Ollama
- **Containerization**: Docker
