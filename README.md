# Retail-Chat-Agent-Backend

This repository contains the scripts to build and deploy the backend component (API) for serving a retail chat agent that recognizes customer needs via image or text inputs.

The backend logic covers the chat and search routes as well as routes to serve the image/text embedding models and image captioning models - Open-CLIP and BLIP respectively.
